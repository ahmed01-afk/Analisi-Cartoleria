# Analisi-Cartoleria
Progetto: Dashboard Analisi Vendite – Cartoleria Alfa

Sistema di monitoraggio delle performance commerciali (2022-2024) basato su un dataset di circa 99.300 € di fatturato totale.

Punti chiave:
Dati: 6 categorie di articoli (es. Penne a sfera, Matite) e 6 venditori (identificati per colore).

Struttura: Tabelle pivot di riepilogo (per anno, articolo e venditore) abbinate a un registro transazioni dettagliato.

Obiettivo: Identificare rapidamente i prodotti e i venditori "top-performer" per ottimizzare le strategie di vendita, il marketing e la gestione delle scorte.
